# Dataset Download Instructions

Download the UCI Air Quality Dataset from:
https://archive.ics.uci.edu/ml/datasets/Air+Quality

Or direct download link:
https://archive.ics.uci.edu/ml/machine-learning-databases/00360/AirQualityUCI.zip

Extract and place the `AirQualityUCI.csv` file in this directory.

The dataset contains:
- Date and Time information
- CO(GT), PT08.S1(CO), NMHC(GT), C6H6(GT), PT08.S2(NMHC), NOx(GT), PT08.S3(NOx), NO2(GT), PT08.S4(NO2), PT08.S5(O3), T, RH, AH sensor readings
- Missing values marked as -200.0
